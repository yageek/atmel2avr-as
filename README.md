
# atmel2avr-as

Tool helping to convert assembly include file provided by ATMEL Studio to be used with GAS through `avr-as`.

## Installing

```
go get github.com/yageek/atmel2avr-as
```

## Usage

```
Usage of atmel2avr-as:
  -F	Overwrite output
  -input string
    	The file to convert
  -output string
    	The output file
```
